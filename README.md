1. **HTML headings**
  - `<h1></h1>`
    # Heading 1
  - `<h2></h2>`
    ## Heading 2
  - `<h3></h3>`
    ### Heading 3
  - `<h4></h4>`
    #### Heading 4
  - `<h5></h5>`
    ##### Heading 5
  - `<h6></h6>`
    ###### Heading 6

2. **HTML paragraph**
  - `<p></p>` - Defines a paragraph

3. **HTML Text Formatting Elements**
  - `<b>` - Bold text
  - `<strong`> - Important text
  - `<i>` - Italic text
  - `<em>` - Emphasized text
  - `<mark>` - Marked text
  - `<small>` - Smaller text
  - `<del>` - Deleted text
  - `<ins>` - Inserted text
  - `<sub>` - Subscript text
  - `<sup>` - Superscript text
